# Typed Array

## Types of typed array
1. Int8Array = -128 to 127
2. Uint8Array = 0 to 255
3. Uint8ClampedArray = 0 to 255
4. Int16Array = -2^15 to 2^15 - 1
5. Uint16Array
6. Int32Array = -2^31 to 2^31 - 1
7. Uint32Array
8. Float32Array = `1.2*10^-38` to `3.4*10^1038`
9. Float64Array
